I’m Tyler Bui-Palsulich, a software engineer and programmer.

About

I am from Orange County, California. I graduated with my Bachelor’s degree from NYU in Computer Science and Math in May 2014 and my Master’s (also from NYU) in Computer Science in May 2015.
I’ve been a software engineer at Google since July 2015.

Projects

NYU Bus Tracker

Android and iPhone application used to track and schedule NYU Buses in real time through Transloc API integration. The Android source code is available on GitHub.

Apache Tika

I am a committer and Project Management Committee member for Apache Tika. Tika is a content analysis and extraction toolkit, supporting over 1400 different file formats.

Apache OODT

I am a committer and Project Management Committee member for Apache OODT (Object Oriented Data Technology). OODT is a data management and archiving system used extensively by NASA JPL.

Auto Ack

An IRC bot created to respond to messages in the JPL XDATA team chatroom. Now has plugins for Facebook, GitHub, and Twitter.

HACS Sublime Text Plugin

Sublime Text plugin for the HACS compiler generator. The plugin handles code completion and syntax highlighting.

cSplash@NYU

cSplash is a one day lecture series at NYU for motivated high school students. In 2013, I taught how an infection spreads through a population (pdf). Here is the MATLAB program I used to model the spread — you can see the number of people susceptible, infected, recovered, and dead. In 2014, I gave an introduction to Git and GitHub. In 2015, I modeled the spread of Ebola through Africa.
